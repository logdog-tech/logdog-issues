# Logdog.tech Issues

## 关于Logdog.tech

Logdog.tech是一款基于浏览器的日志分析工具，无需安装，支持拖拽上传大型日志文件并自动解压。提供智能崩溃点标记、ANR检测、多关键词搜索和正则表达式查询功能，所有分析在本地进行，保护数据隐私。
